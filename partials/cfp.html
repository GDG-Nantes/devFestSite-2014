Call For Papers


Le Call For Papers est a présent fermé !

Rendez vous le 29 Septembre 2014 pour découvrir le programme complet.

Les conférences doivent durer une heure maximum, questions / réponses incluses. Il n'y aura pas de codelabs cette année.
Les thèmes retenus pour cette année sont :

  • Web : Ce track sert pour les sessions traitant de sujets orientés web (Frameworks / générateurs / nouveautés HTML5 / langage (Dart) / animations CSS3...)
  • Cloud : Ce track sert pour les sessions traitant de sujets orientés backend (Google Cloud Platform / Big Data & NoSQL (hadoop, mongo, cassandra) / Déploiement (docker) / Langages serveurs...)
  • Mobile : Ce track sert pour les sessions traitant de sujets autour de la mobilité (Bonnes pratiques / Android / OS Alternatifs / Web mobile / problématiques mobiles...)
  • Découverte : Ce track sert pour les sessions traitant de sujets moins techniques mais offrant un nouveau regard sur d'autres aspects de l'informatique d'aujourd'hui (Design Thinking / UX / Disruption / Nouveaux usages...)
