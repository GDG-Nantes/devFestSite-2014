Call For Papers


Le Call For Papers est ouvert à tous ceux qui souhaitent tenter leur chance en tant que conférencier pour le DevFest Nantes 2014.

Le CFP fermera ses portes le 15 Septembre 2014 à 23h59.

Les conférences doivent durer une heure maximum questions, réponses incluses. Il n'y aura pas de codelabs cette année.
Les thèmes retenus pour cette année sont :

  • Web : Ce track sert pour les sessions traitant de sujets orientés web (Frameworks / générateurs / nouveautés HTML5 / ...)
  • Cloud : Ce track sert pour les sessions traitant de sujets orientés backend web (Google AppEngine / Google ComputeEngine / Langages serveurs / Architectures orientées cloud / ...)
  • Mobile : Ce track sert pour les sessions traitant de sujets autour de la mobilité (Bonnes pratiques / Android / OS Alternatifs / problématiques mobiles / ...)
  • Découverte : Ce track sert pour les sessions traitant de sujets moins techniques au sens développement mais pouvant offrir un nouveau regard sur des aspects important de l'informatique (Design Thinking / UX / Disruption / Nouvelles interfaces / ...)

Pour soumettre votre idée, rien de plus simple, il vous suffit de cliquer sur le bouton suivant : Proposer un sujet
